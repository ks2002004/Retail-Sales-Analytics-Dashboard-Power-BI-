# Retail-Sales-Analytics-Dashboard-Power-BI-
Retail Sales Analytics Dashboard (Power BI)

**Description:**
This project analyzes retail sales data from 2015 to 2017 using Power BI. It provides insights into sales performance, customer behavior, product trends, and returns.

**Key Features:**
Interactive Power BI dashboard for analyzing retail sales performance
Integrated multiple datasets including Sales, Customers, Products, Returns, and Calendar
Implemented data modeling using star schema for efficient querying
Built dynamic KPIs such as Total Revenue, Total Orders, and Return Rate
Enabled time-based analysis using date slicers (Year/Month level insights)
Created category-wise and subcategory-wise performance views
Added geographical analysis using map visual (country-level insights)
Designed interactive filters and slicers for user-driven exploration
Highlighted top-performing products based on orders and revenue
Included trend analysis (weekly revenue & returns) with forecasting

**Key Highlights**
Built an end-to-end BI solution from raw Excel data to interactive dashboard
Applied data cleaning and transformation using Power Query
Designed a scalable data model (fact + dimension tables)
Used DAX measures to calculate KPIs and business metrics
Focused on business insights rather than just visuals
Improved dashboard usability with clean UI, color themes, and layout design
Delivered actionable insights like:
High-performing product categories
Regions contributing maximum revenue
Products with high return rates
Sales trends over time

**Tools Used:**
Power BI
Excel
Power Query
DAX
